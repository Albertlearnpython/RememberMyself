# 首页最终视觉稿说明

## 当前定稿版本

首页当前定稿可概括为：

- 左侧个人档案卡
- 右侧多彩介绍主卡
- 下方模块索引
- 五条记忆溪流
- 最近更新
- 页尾摘录

这已经是当前首页的正式口径。

## 当前视觉主旨

首页追求的是：

**心灵极简主义 + 多彩现代感**

拆开说就是：

- 结构极简
- 留白克制
- 颜色丰富但有品位
- 表面像玻璃和矿石，不像纸页模板

## 首屏最终规则

### 左侧个人档案卡

- 圆角：`32px`
- 表面：浅色玻璃感
- 标签：低饱和彩色胶囊
- 时间线圆点：蓝到粉的渐变

### 右侧介绍主卡

- 圆角：`32px`
- 主底：深蓝黑
- 顶层：多色极光
- 大标题：彩色渐变字
- Motto：独立成块，像题签
- 统计卡：3 张深色卡，每张不同色顶线

## 按钮最终规则

| 类型 | 样式 |
| --- | --- |
| 主按钮 | 蓝 → 紫 → 青多彩渐变 |
| 次按钮 | 半透明玻璃按钮 |
| 按钮圆角 | `14px` |

## 模块索引最终规则

- 桌面端 `3 列`
- 卡片高度更紧凑，不空
- 左下角保留彩光半圆
- 仍然以浅色为主，不与 Hero 抢戏

## 五条溪流最终规则

- 流带容器：浅色珍珠面
- 顶部细线：用流带主色做轻强调
- 卡片本体：深色矿石面
- 卡片高光：对应流带主色做光晕
- hover：卡片轻抬，封面轻微放大

## 首页正式锚点

- `/`：首页
- `/#module-index`：模块索引
- `/#memory-streams`：记忆溪流
- `/#recent-updates`：最近更新

## 手机端最终规则

- 首屏改单列，但人物信息不能缺
- 按钮改纵向堆叠
- 统计卡改单列
- 流带保留，不取消
- 各区块圆角与内边距同步缩小

## 当前结论

首页不再是“浅黄档案页”。

它现在是：

**一张有现代彩光、有结构骨架、有内容流动感的个人总索引封面。**
