# 首页高保真说明

## 当前版本主题

首页当前高保真主题是：

**流彩档案台**

它的重点不是“炫”，而是：

- 一眼能认出主人是谁
- 一眼能看出网站有层次
- 颜色丰富，但仍然克制

## 当前第一屏构成

第一屏已经固定为双栏：

- 左栏：个人档案卡
- 右栏：介绍主卡

这套结构已经替代旧版的大留白 Hero。

## 色彩与材质

### 背景

- 主底：冷白蓝灰
- 局部：低透明度的蓝 / 青 / 紫 / 橙 / 粉光斑
- 目标：通透、现代、不发黄

### 左侧个人档案卡

- 主材质：浅色玻璃卡
- 叠加：轻微彩色柔光
- 观感：像被打磨过的半透明档案夹

### 右侧介绍主卡

- 主材质：深色极光卡
- 叠加：蓝 / 青 / 紫 / 橙 / 粉五色光晕
- 观感：像深色矿石上有流动的彩光

### 统计卡

- 统一深底
- 每张卡各有一条不同颜色的细顶线
- 目的是在同一块区域里制造多彩节奏

## 字体关系

- 中文标题：宋体系衬线
- 正文：干净无衬线
- 元信息：更轻、更小、更窄

要求：

- 标题要稳
- 正文要清
- 元信息要轻

## 当前文字层级

| 层级 | 要求 |
| --- | --- |
| Hero 大标题 | 有气场，但不能过满 |
| 副文案 | 说明网站用途 |
| Motto | 独立成块，像题签 |
| 统计卡标题 | 小、轻、次要 |
| 统计卡数字 | 大、稳、清楚 |

## 按钮规则

- 主按钮：深色多彩渐变
- 次按钮：浅玻璃面或深卡上的半透明按钮
- 同一屏按钮不超过 3 个

当前按钮职责：

- 主按钮：进入藏书室
- 次按钮：查看记忆溪流
- 第三个按钮：查看 GitHub

## 当前视觉禁忌

- 禁止大面积淡黄色底
- 禁止过度纸张感
- 禁止空白大到像未完成页面
- 禁止把首页做成工具台或后台
- 禁止把多彩做成高饱和糖果色

## 当前手机端高保真要求

- 第一屏先看到头像、姓名和按钮
- 右侧介绍主卡在手机端变成完整单列，不可裁断
- 统计卡改为纵向三张
- 流带视窗高度下降，但卡面质感保留
- 不允许为了省空间去掉首页的人物信息

## 当前结论

首页现在的高保真方向已经明确：

**它是一个有多彩层次、有现代质感、但仍然安静克制的个人入口页。**
