# 收支平衡初步设计说明

## 页面定位

收支页不是财务软件后台。

它应该像：

- 日常金钱流向记录页
- 花销与节制的观察页
- 简洁的个人收支账本

## 页面路由

- `/finance`

正式实现时建议使用：`/finance`

## 结构建议

### 桌面端

- 顶部：本周 / 本月概况
- 中间：趋势图
- 下方：流水记录表

### 手机端

- 摘要卡
- 趋势图
- 单列记录卡片

## 核心字段

- 日期
- 金额
- 收入 / 支出
- 分类
- 说明

## 视觉基调

- 克制
- 清晰
- 轻量
- 不要证券交易软件感

## 关键交互

- 快速新增一笔记录
- 过滤收入 / 支出 / 分类
- 查看日 / 周 / 月趋势

## 后续深化方向

- 图表配色
- 列表还是卡片为主
- 手机端快速记账样式
