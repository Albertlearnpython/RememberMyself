# 全站色彩系统方案

## 方案名

静雾档案。

这套色彩系统服务于整个 `RememberMyself`，不是某一个页面的临时配色。

目标是同时满足：

- 心灵的极简主义
- 现代化
- 有设计感，但不浮夸
- 有层次，但不靠高饱和度抢眼

## 核心原则

### 1. 不使用“单一主色统治全站”

全站只共享一套基础中性色骨架。

每个模块再拥有自己的轻微偏色。

### 2. 内容优先于配色

书封、专辑封面、食物照片、景色照片才是最重要的视觉内容。

色彩系统只做秩序、气氛和层次，不和内容争主角。

### 3. 用低饱和度建立高级感

不要高纯度蓝、紫、粉直接铺页面。

整体采用：

- 雾白
- 石墨灰
- 苔绿
- 雾蓝
- 陶土棕

## 首页四条溪流配色

| 流带 | 主色 | 卡片底色倾向 |
| --- | --- | --- |
| 书影流 | `#7B8FA8` | 雾蓝灰 |
| 声纹流 | `#7E7AA8` | 灰紫蓝 |
| 食味流 | `#B9855B` | 柔陶土 |
| 风景流 | `#7A9C92` | 雾青绿 |

要求：

- 流带之间要有差异
- 差异不能靠高饱和度
- 首页整体仍然要统一在浅色、安静的总氛围里

## 组件级落地规则

### 页面背景

- 默认使用 `#F6F3EE`
- 不再使用大面积深蓝黑底

### 卡片

- 主卡片背景：`#FBF8F4`
- 边框：`1px solid #D9D3CA`
- 阴影：轻，避免厚重悬浮感

### 按钮

- 主按钮：苔绿色底
- 次按钮：浅底 + 细边框
- 危险按钮：不做高饱和度红，改为偏陶土棕警示

### 标签

- 标签仍然允许不同色，但都要压低饱和度
- 不出现荧光色系

### 抽屉

- 抽屉背景用二级背景
- 内部表单卡片用卡片主背景
- 通过深浅关系而不是重阴影制造层次

## 第一阶段施工范围

本轮先切换：

- 全站基础 token
- 首页
- 收藏书籍页
- 喜欢的音乐页

其余板块后续沿用同一套系统继续收口。

## 结论

新的色彩系统不是“从深色改浅色”这么简单。

真正要确定的是：

- 全站共享一套雾白中性色骨架
- 各模块带轻微偏色
- 色彩为内容服务
- 让网站像被认真整理过的个人档案册，而不是产品后台或展示型博客
