# 收藏书籍最终视觉稿说明

## 目标

这份文档是收藏书籍页的最终视觉规格说明。

它的作用是让前端在不再反复猜测样式的前提下，直接落版、落组件、落状态。

目标：

- 明确页面主题 token
- 明确列表、详情、文件区、抽屉的最终尺寸和样式
- 明确桌面端与手机端布局
- 明确权限状态下的视觉差异

## 页面路由

- 收藏书籍首页：`/books`
- 书籍详情高亮：`/books/:id`
- 书籍编辑态：仍留在 `/books`，通过抽屉打开，不切换独立路由

## 桌面端布局

### 外层容器

- 页面最大宽度：`1440px`
- 内容区域：`1240px`
- 左右安全留白：`48px`

### 三栏布局

- 左侧筛选栏：`220px`
- 中间书架列表：`1fr`
- 右侧详情区：`380px`
- 栏间距：`24px`

### 抽屉

- 从最右侧覆盖打开
- 宽度：`520px`
- 背景：`#F7F2EA`
- 左侧分割线：`1px solid var(--rm-books-line)`

## 手机端布局

- 页面左右留白：`16px`
- 顶部工具栏高度：`56px`
- 书籍卡片单列
- 详情改为全屏页
- 编辑改为全屏编辑页优先，次选底部抽屉

## 区块视觉说明

- 中：搜索框
- 右：新增书籍按钮

- 圆角：`12px`
- 背景：`rgba(248,244,238,0.9)`
- 边框：`1px solid var(--rm-books-line)`
- 左侧搜索图标
- 占位文字颜色：`var(--rm-books-text-muted)`

### 新增按钮

- 高度：`42px`
- 圆角：`10px`
- 背景：`var(--rm-books-accent-green)`
- 文字：`#F8F4EE`
- 字重：`500`

## 2. 左侧筛选栏

### 样式

- 背景：透明
- 不做厚卡片
- 每组筛选之间间距：`20px`

### 筛选组标题

- 字号：`12px`
- 字重：`600`
- 字母/字间距：`0.03em`
- 颜色：`var(--rm-books-text-muted)`

### 筛选项

- 高度：`32px`
- 内边距：`0 10px`
- 圆角：`999px`
- 背景：透明
- 边框：`1px solid var(--rm-books-line)`

选中态：

- 背景：`var(--rm-books-accent-green-soft)`
- 边框：`rgba(47,75,60,0.22)`
- 文字：`var(--rm-books-accent-green)`

## 3. 中间书架列表

### 列表排布

- 第一版使用纵向列表，不用瀑布流
- 每项最小高度：`132px`
- 项间距：`14px`

### 单本书卡片结构

左：

- 封面图

中：

- 书名
- 作者
- 状态 + 字数 + 标签
- 一句短评

右：

- 更新时间
- 轻箭头

### 书卡尺寸

- 内边距：`18px`
- 背景：`var(--rm-books-surface)`
- 边框：`1px solid var(--rm-books-line)`
- 圆角：`16px`
- 封面尺寸：`72px x 96px`
- 封面比例：`3:4`

### 书卡 hover

- 整卡可点
- 上移 `2px`
- 阴影变为 `var(--rm-books-shadow-hover)`
- 边框色变为 `rgba(107,79,58,0.24)`

### 状态标签颜色

- 在读：背景 `rgba(66,88,74,0.10)`，文字 `#42584A`
- 读完：背景 `rgba(106,83,61,0.10)`，文字 `#6A533D`
- 想读：背景 `rgba(123,116,105,0.10)`，文字 `#7B7469`
- 重读：背景 `rgba(47,75,60,0.14)`，文字 `#2F4B3C`

### 标签色阶

- 标签不手工配置固定颜色
- 前端根据标签名做稳定映射，让同名标签始终保持同一色调
- 推荐至少准备 6 组低饱和色阶，保证同屏出现多个标签也不刺眼
- 字数可使用独立的中性色信息胶囊，不混入标签配色体系

## 4. 右侧详情区

### 区块样式

- 背景：`rgba(248,244,238,0.75)`
- 边框：`1px solid var(--rm-books-line)`
- 圆角：`18px`
- 内边距：`24px`
- 最小高度：与中间列表首屏对齐

### 内容顺序

1. 封面大图
2. 书名
3. 作者 / 译者 / 出版信息
4. 状态 / 字数 / 标签 / 评分
5. 为什么重要
6. 长笔记
7. 阅读时间线
8. 文件区

### 封面大图

- 尺寸：`132px x 176px`
- 圆角：`12px`
- 阴影：极弱 `0 12px 24px rgba(34,31,27,0.08)`

### `为什么重要` 区块

这个区块必须视觉上高于普通长笔记。

建议：

- 顶部小标题 `12px / 600`
- 正文 `16px / 1.8 / 500`
- 左侧一条 `2px` 暗胡桃木竖线

## 5. 受保护文件区

### 区块样式

- 顶部与详情内容分隔 `1px` 细线
- 内边距顶部 `20px`
- 背景不单独做重卡

### 游客态

- 显示文件类型
- 显示文件大小
- 显示 `登录后可下载`
- 按钮置灰不可点

### Viewer 态

- 显示一个主要动作按钮：
  - `下载文件`

### Editor / Owner 态

- 在 Viewer 态基础上增加：
  - `替换文件`
  - `修改可见性`
  - `删除文件`

### 按钮样式

主动作：

- 背景：`var(--rm-books-accent-green)`
- 文字：`#F8F4EE`
- 高度：`40px`
- 圆角：`10px`

次动作：

- 透明背景
- `1px solid var(--rm-books-line)`

危险动作：

- 背景：透明
- 边框：`1px solid rgba(122,98,72,0.24)`
- 文字：`var(--rm-books-accent-walnut)`

## 6. 抽屉最终样式

### 外观

- 宽度：`520px`
- 顶部固定标题栏：`68px`
- 底部固定操作栏：`76px`
- 中间内容滚动

### 标题栏

- 标题：`新增书籍` 或 `编辑书籍`
- 右上：关闭按钮
- 下边界：`1px solid var(--rm-books-line)`

### 内容区

- 内边距：`20px 24px 24px`
- 表单字段间距：`16px`
- 分组间距：`24px`

### 输入框样式

- 高度：`42px`
- 多行文本最小高度：`120px`
- 圆角：`10px`
- 边框：`1px solid var(--rm-books-line)`
- 聚焦态边框：`rgba(47,75,60,0.32)`
- 聚焦态阴影：`0 0 0 3px rgba(47,75,60,0.08)`

### 标签选择器

- `已有标签` 不再整块平铺展示
- 默认折叠为一条摘要输入条，显示当前已选标签概况
- 点击后展开为可搜索的多选面板
- 展开面板内标签区最大高度建议控制在 `260px` 左右，超出后内部滚动
- 新增标签仍保留独立输入框，避免与已有标签选择混在一起
- 目标是让抽屉在标签越来越多时仍然保持安静、紧凑、可扫读

### 底部操作栏

- 左：取消
- 右：保存
- 保存按钮固定宽度：`96px`

## 7. 手机端最终样式

### 列表页

- 搜索框宽度占满
- 筛选改成横向可滚动 chips
- 封面尺寸缩为 `56px x 74px`
- 书名最多两行
- 短评最多两行

### 详情页

- 顶部固定返回和编辑
- 封面居上
- `为什么重要` 和 `长笔记` 改为纵向堆叠
- 文件区按钮改为纵向两行或两列自适应

### 编辑页

- 第一版优先全屏编辑页
- 标题栏固定
- 底部保存按钮固定

## 动效规范

- 列表卡片 hover：`160ms ease`
- 抽屉打开：右向左滑入 `220ms`
- 手机详情进入：轻推入，不使用重遮罩动画
- 不做翻页感动画

## 前端实现验收标准

- 页面第一眼必须像“私人阅读室”，不是文件后台
- 书卡必须整卡可点
- `为什么重要` 必须明显区别于普通元数据
- 游客态文件区要优雅，不要警告框化
- 抽屉内容很多时，顶部和底部操作区不能滚走
- 手机端详情与编辑必须明显好用，不得强塞三栏布局

## 本版结论

这一版已经把收藏书籍页推进到“最终视觉稿说明”层级：

- 颜色
- 字重
- 卡片样式
- 详情结构
- 权限状态
- 抽屉样式
- 手机端样式

前端后续可以直接按这份规格实现第一版。
